Définition
L'algèbre de Boole, ou calcul booléen, est la partie des mathématiques qui s'intéresse à une approche algébrique de la logique, vue en termes de variables, d'opérateurs et de fonctions sur les variables logiques, ce qui permet d'utiliser des techniques algébriques pour traiter les expressions à deux valeurs du calcul des propositions.
Les bases

L'ensemble des valeurs possibles: B = {1, 0}. Par la suite on considère {a, b, c} ⊂ B

L'opérateur . (et): avec a.b = 1 ssi a = 1 et b = 1.

L'opérateur + (ou): avec a+b = 0 ssi a = 0 et b = 0.

L'opérateur ¬ (non): avec ¬a = 1 si a = 0 et ¬a = 0 si a = 1

Associativité

(a+b)+c = a+(b+c) = a+b+c

(a.b).c = a.(b.c) = a.b.c
Commutativité

a+b = b+a

a.b = b.a
Distributivité

a.(b+c) = a.b+a.c

a+(b.c) = a+b.a+c
Idempotence

a+a = a

a.a = a
Elements neutres

a+0 = a

a.1 = a
Absorption

a+1 = 1

a.0 = 0
Théorème de Morgan

¬(a+b) = ¬a.¬b

¬(a.b) = ¬a+¬b

Simplifier

a+a.b = a.1+a.b = a.(1+b) = a.1 = a

a+¬a.b = (a+¬a).(a+b) = 1.(a+b) = a+b

Javascript

Typage dynamique et faible

Les différents types primitifs:

  • boolean: true et false
  • number: Pour les nombres entiers ou décimaux
  • string: Pour les chaînes de caractères
  • null: pour la valeur nulle
  • undefined: pour les valeurs non définies

Le type Object

Truthy/Falsy

Les seules valeurs qui sont évaluées à false sont les suivantes:

  • false
  • 0
  • "" (chaîne vide)
  • null
  • undefined
  • NaN

Toutes les autres valeurs sont évaluées à vrai

Opérateur booléen

Soit quatres variables, falsy, truthy, a et b:

  • b = falsy && a ⇨ b === falsy
  • b = truthy && a ⇨ b === a
  • b = falsy || a ⇨ b === a
  • b = truthy || a ⇨ b === truthy

Exemples d'utilisation:

  • b = monObjet || {}
  • b = monTableau || []
  • b = !!maVariable

Truthy ou falsy ?

"0" && 0 = 0 => falsy

"0" || 0 = "0" = truthy

undefined || {} && 0 || "0" && [] = [] = truthy
